# Recicla API

Backend del sistema de gestión de cooperativa de reciclado.

> Ver [README raíz](../../README.md) para instrucciones de instalación y configuración general.

## Desarrollo

```bash
npm run start:dev -w apps/api
```

## Tests

```bash
npm run test -w apps/api
```

## Documentación

Swagger disponible en `/api/docs` al iniciar el servidor.
